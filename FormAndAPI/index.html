News Page

Fill out the following fields to add a new post to the News page. The expected format for each field is as follows:

  • Title: The name of the blog post i.e. "Making Trails Happen"
  • Type: The type of post i.e. "Press Release"
  • Link: The URL for the post content i.e. "https://www.dvrpc.org/trails/regionaltrailsprogram/"
  • Image: The name of the background image for the post i.e. "sampleImage2.png"
    • Note: TBD on where you should put the images b/c I'm not sure what files/directories you all do or don't have access to.
  • Blurb: The text description for the post. No real limit on length but should be at least XXXXXXX words long.
